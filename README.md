# ASCIIequipotential

Tested on ubuntu, you will need a terminal responsive to \e[XA\r kind of combinations of special characters, as I use them to clear the canvas.

To compile and run:
$ g++ Source.cpp -Ofast
$ ./a.out 200 100 100 10.0f
